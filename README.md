# DevOps

## Vagrant

### Pre-req

- https://www.virtualbox.org/

### Install and Setup

* Plugin vbguest

```sh
vagrant plugin install vagrant-vbguest
```

### Commands

* Criar vagrant file

```sh
vagrant init
```

```sh
vagrant ssh
vagrant halt
vagrant suspend
vagrant destory
vagrant reload
```
